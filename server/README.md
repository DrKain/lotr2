# core2
The core for Land of the Rair.

## Requirements

* Node.js 12+
* MongoDB (or mLab - easier)

## Installation

* `npm install`
* `npm run setup`
* `npm start`

## Environment Variables

Put any environment variables in a [`.env`](https://github.com/motdotla/dotenv) file.

* `DATABASE_URI` - the path to your mongodb install
